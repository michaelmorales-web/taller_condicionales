# EJERECICIO_4: ultimo_digito
- Programa en Python para determinar si los dos ultimos digitos de un número entero son iguales o no

## Análisis

### Variables de entrada
- n: número entero

### Procesamiento
- n1 = (n//100)%10
- n2 = (n//1000)%10

$n1==n2$

### Variables de salida
- iguales
- no iguales
